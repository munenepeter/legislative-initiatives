Dashboard Legislative Initiatives Calendar

  • 2021:01 - Set-up & Core Features

    Jul 06, 2021

    This is the set up of the tool using bootstrap specifically bootswatch , LIs were taken from the sheets provided, formatted using excel, then converted to a Single JSON file using an online tool . After getting the file I researched on how to search through a JSON file using JavaScript as the language is readily avilable and does not require hosting.
    As of version 2021:01 the tool does not incorporate all the available LIs

  • 2021:02 - Design & Look

    Jul 07, 2021

    This saw improvements on the design and feel of the tool and the addition of the version page which explains all the versions of the tool. The tool still needs to include all the LIs provided.
    As of version 2021:02 the tool does not incorporate all the available LIs

  • 2021:03 - Move to Vue Js

    Jul 08, 2021

    The tool was moved from using pure(Vanilla) JavaScript to using Vue js of the tool and this version removed the use of an external js file, (the file will not be deleted though)
    As of version 2021:03 the tool does not incorporate all the available LIs

  • 2021:04 - Updated the API

    Aug 05, 2021

    This version saw the Updating of the API to accomodate all the LIs which do not have either the abbrevations or the names. The version also produces another version of the tool but in a lighter interface which was implemented using the famous Jquery js
    As of version 2021:04 the tool incorporates all the available LIs
    The next update might see revamping of the interface and inclusion of a shortcut to copy the LI searched
